# NHS Step Up Career Challenge  
## Turning Data into Insight: Building Awareness of Mental Health Prescribing

This repository contains my completed work for the NHS Step Up Career Challenge organised by DigData and the NHS. The project explores NHS antidepressant prescribing data using Excel, R, and Python to analyse prescribing trends, regional variation, prescribing volumes, and prescribing costs across England.

---

# Project Overview

The NHS is experiencing increasing demand for mental health services alongside rising antidepressant prescribing rates. This project investigates prescribing data to identify patterns, trends, and insights that could help support healthcare decision making and improve patient care.

The work completed in this repository follows the three NHS Step Up challenge pathways:

- Challenge A вЂ“ Excel Analysis
- Challenge B вЂ“ R Analysis
- Challenge C вЂ“ Python Analysis

Each challenge was completed using:
- Excel
- R
- Python

This allowed comparison of different analytical approaches, workflows, and visualisation techniques across multiple platforms.

---

# Objectives

The analysis aimed to:

- Explore antidepressant prescribing trends
- Identify the most prescribed antidepressants
- Analyse prescribing costs nationally and regionally
- Compare prescribing volume against prescribing spend
- Create data visualisations and reports
- Present findings clearly for non-technical audiences
